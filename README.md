## Blogg.

Blogg is a web application where you can share your knowledge and ideas.

## Run

Clone this repository
```bash
git clone https://github.com/kmr-ankitt/Blogg
cd Blogg
```

Install all required packages
```bash
npm i
```

Start the server
```bash
npm index.js
```
